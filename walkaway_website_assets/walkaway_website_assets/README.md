# WalkAway / Siuil Leat Website Asset Pack

This folder contains the practical assets needed to create a simple one-page website from the ambassador brochure.

## Folder contents
- assets/images/ - extracted images and brochure previews
- assets/brand/brand-guide.md - colours, typography and design direction
- assets/copy/website-copy.md - draft website copy by section
- assets/copy/site-map.md - recommended website structure
- assets/css/styles.css - starter CSS variables and layout styles

## Recommended minimum website build
Use a one-page structure:
Hero, About, Vision, Four Pillars, Co-founders, How You Can Help, Contact.

## Image notes
- walkaway-logo-card.png - usable logo image, best for footer/contact area unless a clean transparent logo is supplied.
- walkaway-icon.png - icon/mark for favicon or small brand use.
- sharon-lynn-headshot.jpeg - co-founder image.
- bryan-gallagher-headshot.png - co-founder image.
- walkaway-four-pillars-infographic.png - source infographic; for the website, recreate these as responsive HTML cards rather than relying only on the image.
